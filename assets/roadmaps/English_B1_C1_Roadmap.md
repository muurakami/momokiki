# REFACTORING ENGLISH
## From B1 to C1 for AI/ML Engineers
### A Task-Based Language Immersion Roadmap

## EXECUTIVE SUMMARY
This roadmap transforms English from B1 to working C1 through realistic AI/ML communication tasks.

## PHASE OVERVIEW
### Phase 1 - Unblock Speaking
- Standup simulator - daily short recordings
- Bug triage - explain symptoms, hypothesis, plan

### Phase 2 - Technical Fluency
- Architecture walkthroughs - defend trade-offs
- Code review sparring - push back politely

### Phase 3 - Interview and C1
- Behavioral library - STAR stories
- Mock interviews - technical and behavioral
