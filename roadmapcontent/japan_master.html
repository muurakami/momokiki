УЧЕБНЫЕ МОДУЛИ

学習モジュール
МОДУЛЬ 01
Хирагана
ひらがな
Основная слоговая азбука. Как в Dark Souls — нельзя победить финального босса, не освоив базовые атаки. Каждый символ — один слог.
  • 46 основных символов (а-ряд до ва-ряд)
  • Дакутен が、ざ、だ、ば + хандакутен ぱ
  • Малые символы ゃゅょ (комбинации)
  • Тренировка: RTKana, Duolingo, Anki
  • Метод: мнемоника + написание от руки
НАЧАЛЬНЫЙ · 2 нед.
МОДУЛЬ 02
Катакана
カタカナ
Азбука для иностранных слов. Именно здесь пишется アニメ (аниме), ゲーム (гэему), ナルト (Наруто). Выучи и читай слова из своего мира!
  • 46 символов — те же звуки, другое начертание
  • Гайрайго — иностранные заимствования
  • コーヒー、パン、テレビ — узнаёшь?
  • Тренировка: читай меню в аниме!
  • Инструмент: Real Kana, Katakana Game
НАЧАЛЬНЫЙ · 2 нед.
МОДУЛЬ 03
Грамматика N5–N4
文法の基礎
Японская грамматика — как боевая система в RPG. SOV порядок меняет всё. Частицы — как теги в программировании: помечают роль каждого слова.
  • Частицы: は・が・を・に・で・へ・と・の・も・か
  • Глаголы: группы 1, 2, 3 (る、う、する/くる)
  • Время: настоящее/будущее vs прошедшее
  • Вежливость: です/ます форма
  • Источник: Genki I&II, Tae Kim Grammar
НАЧАЛЬНЫЙ · 2–3 мес.
МОДУЛЬ 04
Кандзи
漢字
2136 иероглифов — не так страшно, как кажется. Это как прокачка скиллов: по 10–15 в день через SRS-систему. Через год читаешь манга, через два — книги.
  • N5: 80 кандзи — 日、月、水、火、山、川...
  • N4: 180 кандзи — базовые глаголы и существительные
  • N3: 300+ кандзи — можно читать манга
  • RTK метод — запоминание по историям
  • WaniKani — мнемоника + SRS системы
СРЕДНИЙ · 6–18 мес.
МОДУЛЬ 05
Говорение
会話
Язык живёт в разговоре. Как прокачка персонажа через битвы — без практики с живыми людьми не вырастешь. iTalki, HiNative, японские Discord-серверы.
  • Shadowing — повторяй за носителями
  • Anki + аудио — тренируй уши
  • iTalki / Preply — репетиторы из Японии
  • Пиши в японских чатах (mixi, Twitter JP)
  • Языковой обмен (tandem partners)
СРЕДНИЙ · постоянно
МОДУЛЬ 06
Кейго — вежливость
敬語
В Японии вежливость — это не просто слова, это код чести самурая. Три уровня: 丁寧語 (вежливо), 尊敬語 (уважение к собеседнику), 謙譲語 (унижение себя).
  • 丁寧語 — です/ます форма (уже знаешь!)
  • 尊敬語 — いらっしゃる вместо いる/ある/行く
  • 謙譲語 — まいる、おる、いたす
  • Деловые фразы: お世話になっております
  • Нужен для JLPT N3 и выше
ПРОДВИНУТЫЙ · 2–3 мес.

УЧИСЬ ЧЕРЕЗ АНИМЕ И ИГРЫ

アニメと遊びで学ぶ
Лучший способ выучить язык — полюбить его носителей. Аниме, манга, JRPGs и визуальные романы дадут тебе тысячи часов живого японского. Герои аниме буквально учат тебя говорить.
🍥
NARUTO / НАРУТО
Идеально для начинающих! Речь чёткая, эмоциональная. Наруто говорит простыми предложениями. Учи слова упорства и дружбы — основа японской культуры.
信じてもらえなくても、絶対に諦めない！
Даже если никто не верит — я никогда не сдамся!
⚔️
DEMON SLAYER / КНБ
Самурайская лексика, формальная речь, архаичные выражения. Тандзиро говорит очень вежливо — прекрасный образец です/ます формы в действии.
俺は長男だから我慢できる！
Я старший брат, поэтому я могу терпеть!
🎮
FINAL FANTASY / JRPG
Меню игр на японском — сотни слов в контексте. FF7, Persona, Dragon Quest — ставь яп. интерфейс. Визуальные романы — лучший способ читать.
攻撃する / 魔法を使う / 逃げる
Атаковать / Использовать магию / Сбежать
📚
МАНГА ДЛЯ УЧЁБЫ
Начни с: Yotsuba&! (самая простая), Chi's Sweet Home, Shirokuma Cafe. Затем: One Piece, Naruto с JP субтитрами. Furigana помогает читать кандзи.
よつばと！— идеально для N5–N4
Ежедневная жизнь девочки · Живой разговорный язык
🎵
J-POP / АНИМЕ OST
Гимны Naruto, Demon Slayer, Your Lie in April. Учи тексты песен — мозг запоминает через музыку в 5 раз быстрее. YOASOBI, LiSA, Yorushika.
紅蓮華 — LiSA (Demon Slayer OP)
Каждое слово — живой язык в ритме
📖
РАНОБЭ / LIGHT NOVELS
Sword Art Online, Re:Zero, Oregairu — те самые книги, которые стали аниме. Сначала читай с фуриганой. Используй Yomichan (расширение для браузера).
俺ガイル → NHK Web Easy → 新聞
Путь читателя: ранобэ → новости → газеты

АРСЕНАЛ САМУРАЯ

道具と戦術
🃏
Anki — SRS система
Интервальное повторение — научно доказанный способ запоминания. Скачай колоды: Core 2000/6000, JLPT N5-N1. Делай 10–20 новых карточек в день.
⛩️
WaniKani
Платформа для кандзи по мнемонике. 60 уровней = все Jōyō кандзи. 1–2 года при регулярной работе. Первые 3 уровня бесплатны.
🌐
Bunpro — грамматика
SRS для грамматических конструкций. Охватывает всё от N5 до N1. Объяснения на английском с примерами из аниме. Идеально как дополнение к Genki.
🎧
Shadowing техника
Слушай носителя и немедленно повторяй вслух. Начни с Shadowing: Japanese as a Foreign Language (Кента Минами). 15 мин/день = огромный прогресс.
🔍
Yomichan / Jisho
Yomichan — расширение браузера: наводишь мышку на японское слово → мгновенный перевод. Jisho.org — лучший словарь. Незаменимы при чтении.
📺
Language Reactor
Netflix + Language Reactor = аниме с двойными субтитрами (JP + RU). Кликаешь на слово → мгновенный перевод + добавление в Anki. Бесценно!
